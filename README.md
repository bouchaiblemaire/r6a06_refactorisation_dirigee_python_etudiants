# R6A06 — Customer / Movie / Rental (Python)

Traduction Python du mini-projet Java (Customer/Movie/Rental) destiné à la **revue de code**.

## Créer et activer un environnement virtuel

```bash
python -m venv .venv
```

- Windows :
```bash
.venv\Scripts\activate
```

- Linux/macOS :
```bash
source .venv/bin/activate
```

## Lancer les tests

```bash
python -m pip install -U pip pytest
pytest
```
